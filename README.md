# Practica spam
## De que trata la practica
esta es una practica de un detector de spam en la usa como aprendizaje un archivo scv para detectar que 
mensajes son spam dentro de un correo
## como?
1. creamos una carpeta para el proyecto
2. creamos dentro un entorno virtual
3. creamos una carpeta de resources y otra de data
4. dentro de la carpeta de resources creamos el archivo screen.py y el archivo spam_detector.py
5. dentro el archivo spam_detector se hizo el codigo donde sera la ejecucion de deteccion de spam
6. dentro del acrivo screen se hizo el codigo donde es la parte grafica usando la libreria de tkinter
7. se ejecuto el programa en un entorno virtual y funciono
